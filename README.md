  - A transportation tracking application that allows users to input their location and destinations and receive relevant ttc bus routes from an external API.
#
- a list of the user stories, along with who is responsible for each one
  - User Stories
    1. As a user, I want to choose my location and destination on a map to find the closest bus stop
    2. As a user, I want to see which bus to take to get to my destination.
    3. As a user, I want to see the schedule of the buses for a given bus stop
    4. As a user, I want to see the eta of the buses for a given bus stop
    5. As a user, I want to be able to search any route number and show all the busses/locations of each bus on that route
    6. As a user, I want to see the occupancy status of each bus in real time
    7. As a user, I want to receive alerts for delays or disruptions so that I can plan accordingly.
